TextMagic API v2

TextMagic'u SMS API on platvorm, mille kaudu on võimalik arendada sõnumsiderakendusi kasutades TextMagic'u infrastruktuuri. Selle kaudu saad saata ja vastu võtta SMS sõnumeid, pärida informatsiooni sissetulevate ja väljaminevate sõnumite kohta, hallata kontakte, valmistada formaate, ajastada sõnumite väljasaatmist ning töödelda suurt hulka sõnumeid.

Kaks viisi kuidas kasutada TextMagic API'd: REST API ja Emaili SMS API.
Kooditeegid: Node.js, PHP, Java, Python, .NET, Ruby, Delphi, Go, Perl, C++, Shell ja C#.

Kuidas genereerida API identimisteabe

Sõnumite saatmiseks on sul vaja API võtit. API võtmed to sarnased kasutajaparoolile; erinevus nende kahe vahel on see, et API võti annab ligipääsu vaid API'le: sellega ei saa sisse logida TextMagic'usse.
Sinu rakendus saadab sisselogimisteabe iga API taotlusega HTTP päisena: X-TM-Username on sinu TextMagic'u kasutajanimi ning X-TM-Key on sinu API võti.

Kuidas saada kätte meie API võtit

1. Logi sisse TextMagic'usse.
2. Mine API seadete lehele.
3. Kliki 'Add new API key' nuppu.
4. Sisesta oma rakenduse nimi API võtme jaoks.
5. Kliki 'Generate new key' nuppu.
6. Sinu API võti on olemas.

Kuidas REST API töötab

REST API kasutab HTTP protokolli et saata ja vastu võtta sõnumeid. REST sõnumid on tavaliselt kodeeritud JSON dokumentidena, see on turvalisem kui varaseimad meetodid näiteks XML'il põhinev SOAP protokoll. REST API kasutab samu funktsioone mida brauserid kasutavad: POST, GET, PUT ja DELETE. Need vastavad CRUD funktsioonidele: create, read, update ja delete.

Näide:
GET /api/v2/contacts (Tagastab kõik sinu kontaktid).
GET /api/v2/contacts/{id} (Tagastab teatud kontakti).
POST /api/v2/contacts (Loob uue kontakti).
PUT /api/v2/contacts/{id} (Uuendab olemasolevat kontakti).
DELETE /api/v2/contacts{id} (Kustutab olemasoleva kontakti ära).

See ongi nii lihtne! Kusjuures, ainult seda ongi vaja et hallata kõike oma kontakte TextMagic'u kasutajas.

TextMagic'u Sandbox

Sandbox on tööriist, millega saad testida TextMagic REST API taotlusi ilma ühegi rakenduse alla laadimata või koodi kirjutamiseta.
Et taotleda andmeid oma TextMagic kasutaja kaudu, on sul vaja sisestada oma kasutajanimi ja API võti. Sul on võimalus salvestada need oma brauserisse.
Et vaadata funktsioonide dokumentatsiooni, mille alla käivad parameetrid, filtrid, formaadid ja tagastuskoodid, pead lihtsalt vajutama selle nimele.
Sandboxi kaudu saad testida TextMagic'u REST API'd läbi oma brauseri. Programmeerimisoskusi pole vaja. Et kasutada Sandboxi, lihtsalt vajuta selle käsu peale, mida soovite testida. Sisestage vajalikud parameetrid ja klikkige nupul Try! et esitada taotlust. URL vastab sellele käsule, mida soovite kasutada, näiteks: /api/v2/templates vastab templates käsule. Saate HTTP-vastuse koodi millega koos on JSON response body, mis näitab teile kas taotlus õnnestus voi mitte. HTTP vastuse pealistes leiad lisateavet.

Sõnumite saatmine

Peamine põhjus TextMagic API kasutamiseks on SMS sõnumite saatmine.
Et saata sõnumeid TextMagic API v2'ga, saatke POST taotlus sõnumite URL'ile.
Näide: 'POST https://rest.textmagic.com/api/v2/messages'.

Kuidas saata sõnumit vaid ühele kontaktisikule

Kui tahad saata üksikut sõnumit ühele kontaktisikule, pane kirja vaid text ja phones parameetrid. text peab sisaldama seda sõnumit, mida soovid saata ja phones peab sisaldama kontaktisiku telefoninumbrit. Näiteks: Kui tahad saata SMS sõnumit tekstiga 'I love TextMagic' telefoninumbrile +44(0)7860021130, siis kasutage järgnevat käsku:

POST https://rest.textmagic.com/api/v2/messages

Parameetritega:
text: Hello
phones: 4958888104

Kui saadate sõnumi vähemalt ühele kontaktisikule, siis TextMagic API tagastab sessionId ja messageId. Muide, saad ka vaadata sõnumi teekonda ID'ga. Kuna see ei ole suur sõnumi saadetis ega ka ajakavaga saadetud sõnum, siis bulkId ja scheduleId on null.

Kuidas saata suurt hulka sõnumeid korraga

On võimalik ka saata suurt hulka sõnumeid korraga erinevatele kontaktidele. Oletame, et teil on list ID'ga 55314, milles asuvad 100 kontakti. Kui tahate neile SMS sõnumi saata, tehke järgnev päring:

POST https://rest.textmagic.com/api/v2/messages

Parameetritega
text: Hello
lists: 55314
phones: 44312664788500
contacts: 318348295454
