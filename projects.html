TextMagic API v2

TextMagic'u SMS API on platvorm, mille kaudu on võimalik arendada sõnumsiderakendusi kasutades TextMagic'u infrastruktuuri. Selle kaudu saad saata ja vastu võtta SMS sõnumeid, pärida informatsiooni sissetulevate ja väljaminevate sõnumite kohta, hallata kontakte, valmistada formaate, ajastada sõnumite väljasaatmist ning töödelda suurt hulka sõnumeid.

Kaks viisi kuidas kasutada TextMagic API'd: REST API ja Emaili SMS API.
Kooditeegid: Node.js, PHP, Java, Python, .NET, Ruby, Delphi, Go, Perl, C++, Shell ja C#.

Kuidas genereerida API identimisteabe

Sõnumite saatmiseks on sul vaja API võtit. API võtmed to sarnased kasutajaparoolile; erinevus nende kahe vahel on see, et API võti annab ligipääsu vaid API'le: sellega ei saa sisse logida TextMagic'usse.
Sinu rakendus saadab sisselogimisteabe iga API taotlusega HTTP päisena: X-TM-Username on sinu TextMagic'u kasutajanimi ning X-TM-Key on sinu API võti.

Kuidas saada kätte meie API võtit

1. Logi sisse TextMagic'usse.
2. Mine API seadete lehele.
3. Kliki 'Add new API key' nuppu.
4. Sisesta oma rakenduse nimi API võtme jaoks.
5. Kliki 'Generate new key' nuppu.
6. Sinu API võti on olemas.
